	- detecting hallucinations
		- NLI: Natural Language Inference (or Entailment)
			- given a premise and hypothesis,
			- ex:
				- premise: sun rises in east, sets in west
				- hypothesis: the sun rises in the east
				- provide to NLI model
				- output:
					- entailment (truthful statement): sun rises in east
					- contradiction: sun rises in west
					- neutral: sun is a star
			- use nltk library
			- NLI pipeline is just a set of tests
				- test each case: entailment/contra/neutral
		- can test which sources are retrieved for a given sentence
			- then can check whether sources entail (prove) the sentence
		- zero-shot classification
